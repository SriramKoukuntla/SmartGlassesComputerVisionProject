Software for smart glasses with minimal specs that provides visual assistance for visually impaired individuals.

I am using react frontend to see changes I am doing to OCR, Objection Detection, and Monocular Depth Estimation algorithms in real time. 

Starting Backend:
1. Activate virtual environment: "./venv/Scripts/activate"
2. Start up python program: "python -m app.main"

Starting Frontend:
1. npm install
2. npm run dev
