# Smart Glasses Frontend

A React application for smart glasses that provides visual assistance for visually impaired individuals.

## Features

- Real-time camera feed access
- Front and back camera switching
- Modern, accessible UI
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (typically `http://localhost:5173`)

## Camera Permissions

When you first access the app, your browser will prompt you to allow camera access. Please grant permission to use the camera features.

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.
